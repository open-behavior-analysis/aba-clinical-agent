# 🏗️ ABA therapist胜任力矩阵与晋升路线

> 本文件为 `staff-evaluation` 的参考字典。定义了从trainee生到lead supervisor的完整晋升阶梯、各级胜任力标准和考核维度。

---

## Promotion Ladder Overview

```
Level 1：traineetherapist (Trainee)
    ↓ 考核通过 + 累计 200h 实操
Level 2：juniortherapist (Junior Therapist)
    ↓ 考核通过 + 累计 500h 实操 + RBT 认证（可选）
Level 3：therapisttherapist (Therapist / RBT)
    ↓ 考核通过 + 累计 1500h 实操 + 独立带案能力
Level 4：seniortherapist (Senior Therapist)
    ↓ 考核通过 + 累计 3000h + 带教经验 + BCaBA（可选）
Level 5：assistant supervisor (Assistant Supervisor / BCaBA)
    ↓ 考核通过 + BCBA 认证
Level 6：lead supervisor / 临床总监 (Clinical Director / BCBA)
```

## Competency Dimension Definitions

### 维度 A：ABA 基础理论
| 等级 | 要求 |
|:---|:---|
| L1 trainee | 了解reinforcement/惩罚基本概念，能复述 4 种行为功能 |
| L2 junior | 掌握 MO/EO、prompt hierarchy、DTT 三项结构，能独立解释给家长 |
| L3 therapist | 理解 VB 分类（mand/tact/echoic/intraverbal），能分析简单行为链 |
| L4 senior | 能独立进行 FBA 假设推理，理解competing behavior model |
| L5 assistant supervisor | 能设计基于functional analysis的 BIP，指导他人理解行为学原理 |

### 维度 B：教学技能
| 等级 | 要求 |
|:---|:---|
| L1 trainee | 能在督导指导下执行 DTT（给SD→等待→辅助→reinforcement） |
| L2 junior | 独立执行 DTT，掌握零延迟辅助和 4 步纠错，reinforcement时机 ≤ 0.5 秒 |
| L3 therapist | 独立执行 DTT + NET + IT，能根据孩子状态灵活切换教学形式 |
| L4 senior | 能设计新的教学程序，拆解目标为教学切片，制作Teaching Guide |
| L5 assistant supervisor | 能带教新老师，使用 BST 框架培训教学技能 |

### 维度 C：行为管理
| 等级 | 要求 |
|:---|:---|
| L1 trainee | 能在督导指导下执行预设的行为应对方案 |
| L2 junior | 能独立执行 BIP 中规定的antecedent manipulation和后果管理 |
| L3 therapist | 能准确记录 ABC，发现行为功能的初步线索 |
| L4 senior | 能提出行为功能假设，设计并实施替代行为教学 |
| L5 assistant supervisor | 能独立完成 FBA 全流程，撰写 BIP，指导团队执行 |

### 维度 D：数据记录与分析
| 等级 | 要求 |
|:---|:---|
| L1 trainee | 能按模板准确记录频率/持续时间/正确率数据 |
| L2 junior | 能独立完成课后记录卡的全部填写，数据准确率 ≥ 90% |
| L3 therapist | 能绘制趋势图，判断数据走向（上升/平稳/下降），提出调整建议 |
| L4 senior | 能进行跨目标、跨时段的数据交叉分析，撰写数据报告 |
| L5 assistant supervisor | 能基于数据做临床决策（加速/减速/维持/generalization/终止） |

### 维度 E：家长沟通
| 等级 | 要求 |
|:---|:---|
| L1 trainee | 能礼貌接待家长，不主动给出临床建议（转交督导） |
| L2 junior | 能用大白话向家长反馈今日教学内容和孩子表现 |
| L3 therapist | 能回答家长关于generalization和家庭配合的常见问题 |
| L4 senior | 能独立进行家长培训（BST 模式），教家长在家执行简单程序 |
| L5 assistant supervisor | 能处理家长投诉和冲突，进行专业心理建设 |

### 维度 F：督导与带教能力
| 等级 | 要求 |
|:---|:---|
| L1-L3 | 不适用（被督导者） |
| L4 senior | 能作为"老带新"中的老师角色，给出操作层面的具体指导 |
| L5 assistant supervisor | 能独立使用 BST 培训新老师，能撰写督导反馈报告，能主持Case Conference |
| L6 lead supervisor | 能设计培训课程体系，制定机构级 SOP，进行assistant supervisor的督导 |

## Assessment Method Reference

| 考核方式 | 适用场景 | 说明 |
|:---|:---|:---|
| **笔试** | 理论维度 (A) | 选择题+情境分析题 |
| **实操 BST** | 教学技能 (B) + 行为管理 (C) | 现场示范→角色扮演→反馈 |
| **数据审计** | 数据能力 (D) | 随机抽查课后记录的准确率 |
| **模拟沟通** | 家长沟通 (E) | 模拟家长问答或投诉场景 |
| **督导日志审阅** | 督导能力 (F) | 审阅其带教记录和反馈质量 |
| **综合答辩** | 晋升评估 | 呈现个案完整干预数据+临床推理 |

## Promotion Assessment Passing Criteria

| 目标等级 | 必须mastery维度 | 累计实操时数 | 附加条件 |
|:---|:---|:---|:---|
| L2 junior | A≥L2, B≥L2, D≥L2 | ≥ 200h | 无 |
| L3 therapist | A≥L3, B≥L3, C≥L3, D≥L3 | ≥ 500h | RBT 认证（推荐） |
| L4 senior | 全维度≥L4 | ≥ 1500h | 至少独立带过 3 个个案 |
| L5 assistant supervisor | 全维度≥L5 | ≥ 3000h | BCaBA 认证（推荐）+ 至少带教过 2 名新老师 |
