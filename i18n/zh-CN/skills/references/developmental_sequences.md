## 一、VB-MAPP 里程碑进阶链

> 基于 Sundberg (2008) VB-MAPP 评估框架，每个领域按 Level 1 (0-18月) → Level 2 (18-30月) → Level 3 (30-48月) 列出关键里程碑节点。

### Mand（提要求）
| Level | 里程碑 | 典型课题示例 |
|:---|:---|:---|
| L1-1 | 在动机操作下发出2个词的 mand | 要求"要+物品名" |
| L1-5 | 自发 mand 10个不同强化物 | 提要求10种物品/活动 |
| L2-6 | 缺失物件时自发 mand | "要剪刀"（手工活动中） |
| L2-9 | mand 包含2词组合，20种以上 | "给我+物品"、"打开+物品" |
| L3-11 | mand 他人执行动作 | "帮我开门"、"你来推" |
| L3-13 | mand 获取信息（what/where/who） | "这是什么？"、"在哪里？" |
| L3-15 | mand 使用完整句子，包含形容词/介词 | "我要那个大的红色球" |

### Tact（命名）
| Level | 里程碑 | 典型课题示例 |
|:---|:---|:---|
| L1-1 | tact 2个物品 | 看到物品说出名称 |
| L1-5 | tact 10个物品（无辅助） | 命名常见物品 |
| L2-6 | tact 动作/动词 | 命名"跑、跳、吃、喝" |
| L2-8 | tact 50个物品+10个动作 | 扩展命名库 |
| L2-10 | tact 物品特征（颜色/形状/大小） | "红色的球"、"大象很大" |
| L3-11 | tact 200+词汇含形容词/介词/代词 | 描述性命名 |
| L3-13 | tact 情绪（自己和他人） | 命名"开心/难过/害怕/生气" |
| L3-15 | tact 事件/因果关系 | "他哭了因为摔倒了" |

### Listener Responding（听者反应）
| Level | 里程碑 | 典型课题示例 |
|:---|:---|:---|
| L1-1 | 对名字有反应 | 叫名转头/注视 |
| L1-3 | 执行2步指令 | "把杯子给我" |
| L1-5 | 从4个里选出正确物品，10次+ | 听觉理解-物品辨别 |
| L2-6 | 执行含动词+名词的指令 | "拍球"、"推车" |
| L2-8 | 按特征选择（颜色/形状） | "给我红色的" |
| L2-10 | 执行3步指令 | "拿书放桌上然后坐下" |
| L3-11 | 按功能/特征/类别选择 | "给我可以喝的东西" |
| L3-14 | 理解时间概念相关指令 | "先…再…"、"等一下" |

### Echoic（仿说）
| Level | 里程碑 | 典型课题示例 |
|:---|:---|:---|
| L1-1 | 仿说5个语音 | 模仿元音/辅音 |
| L1-3 | 仿说10个单词 | 模仿常见词 |
| L1-5 | 仿说25个单词（可辨认） | 扩展仿说词库 |
| L2-6~10 | 仿说2-3词短语 | "要喝水"、"妈妈来" |
| L3 | 仿说完整句子（5词+） | 模仿完整句子 |

### Intraverbal（交互式语言）
| Level | 里程碑 | 典型课题示例 |
|:---|:---|:---|
| L1-5 | 填词完成5首儿歌 | "小星星___" → "亮晶晶" |
| L2-6 | 回答10个"什么"问题 | "你用什么喝水？" → "杯子" |
| L2-8 | 回答25个 WH 问题 | what/who/where 混合 |
| L2-10 | 回答关于当下活动的问题 | "你在做什么？" → "画画" |
| L3-11 | 回答涉及功能/特征/类别的问题 | "什么动物会飞？" → "鸟" |
| L3-13 | 讲述个人经历（2-3句） | "今天在学校做了什么？" |
| L3-15 | 逻辑推理/因果回答 | "为什么要穿雨衣？" → "因为下雨" |

### Social（社交行为）
| Level | 里程碑 | 典型课题示例 |
|:---|:---|:---|
| L1-2 | 注视说话者2秒 | 眼神接触 |
| L1-5 | 自发向同伴发出社交行为 | 招手/微笑 |
| L2-6 | 与同伴维持互动30秒 | 轮流玩玩具 |
| L2-10 | 自发请求同伴互动 | "你来玩吗？" |
| L3-11 | 合作游戏（有共同目标） | 一起搭积木 |
| L3-14 | 假想游戏（角色扮演） | 扮演医生/厨师 |
| L3-15 | 多轮会话（4轮+） | 就一个话题持续对话 |

### Imitation（模仿）
| Level | 里程碑 | 典型课题示例 |
|:---|:---|:---|
| L1-2 | 模仿3个大动作 | 拍手/举手/跺脚 |
| L1-5 | 模仿8个动作含精细动作 | 捏/指/竖大拇指 |
| L2-6 | 模仿动作序列（2步） | 先拍头再拍肩 |
| L2-10 | 模仿同伴的游戏动作 | 看到同伴叠高仿做 |
| L3 | 模仿复杂序列（3步+） | 多步手工操作 |

### Play（游戏）
| Level | 里程碑 | 典型课题示例 |
|:---|:---|:---|
| L1-3 | 功能性操作5个玩具 | 用勺子假装喂娃娃 |
| L1-5 | 独立玩耍1分钟 | 自发探索玩具 |
| L2-6 | 2步假想游戏序列 | 倒水→喂娃娃 |
| L2-10 | 与同伴平行游戏5分钟 | 在同一区域各自玩 |
| L3-12 | 角色扮演3步+序列 | 当医生：听诊→打针→贴创可贴 |
| L3-15 | 合作假想游戏 | 与同伴共建场景 |

## 二、ABLLS-R 关键任务序列

> 基于 Partington (2006) ABLLS-R 评估工具，列出常用 Section 的任务进阶顺序。

### Section A — 合作与强化物效力
A1 接受强化物 → A2 在活动中接受强化物 → A3 配合1:1教学 → A4 安坐30秒 → A8 安坐5分钟 → A12 在小组中安坐

### Section B — 视觉表现（配对/分类）
B1 物品-物品配对 → B3 图片-图片配对 → B5 图片-物品配对 → B9 颜色配对 → B12 形状配对 → B15 字母配对 → B19 分类（按类别） → B22 序列排列

### Section C — 听者反应（接受性语言）
C1 对名字有反应 → C3 执行单步指令 → C7 辨别常见物品 → C12 辨别图片 → C16 按特征辨别 → C21 按功能辨别 → C28 执行含介词的指令 → C35 理解复杂句子

### Section D — 模仿
D1 大动作模仿 → D4 物品操作模仿 → D8 精细动作模仿 → D12 口腔动作模仿 → D16 多步动作序列模仿

### Section G — 提要求（Mand）
G1 表示要/不要 → G3 指向想要的物品 → G6 用词语 mand → G10 自发 mand 10种+ → G15 mand 缺失物品 → G20 mand 信息 → G24 mand 用完整句

### Section H — 命名（Tact）
H1 命名5个物品 → H5 命名20个物品 → H10 命名动作 → H15 命名特征 → H20 命名类别 → H25 命名功能 → H30 描述性命名（多特征组合）

### Section J — 交互式语言（Intraverbal）
J1 儿歌填词 → J4 功能性回答 → J8 回答 WH 问题 → J12 类别列举 → J16 描述事件 → J20 逻辑推理对话

### Section K — 自发语言
K1 自发 mand → K4 自发 tact → K8 自发评论环境 → K12 发起对话 → K15 讲述经历

### Section R — 精细动作
R1 抓握物品 → R3 捏取小物品 → R6 穿珠 → R9 用剪刀剪直线 → R12 控笔画线 → R15 描写笔画 → R18 独立书写

## 三、领域专属教学阶梯

> 按常见教学领域整理的典型进阶路径，供课题升级决策时参考。

### 🗣️ 语言领域
```
仿说（语音→单词→短语→句子）
  ↓
提要求（单词 mand → 2词组合 → 缺失物件 mand → mand 信息 → mand 动作 → 完整句 mand）
  ↓
命名（物品→动作→特征→情绪→因果→描述性命名）
  ↓
听者反应（物品辨别→图片辨别→按特征选→按功能选→按类别选→复杂指令）
  ↓
交互式语言（填词→功能问答→WH问答→类别列举→描述事件→逻辑推理→对话）
```

### 📚 学习准备领域
```
安坐（30秒 → 1分钟 → 5分钟 → 小组中安坐）
  ↓
注视（叫名转头 → 注视说话者2秒 → 注视演示5秒 → 跟随视觉指引）
  ↓
模仿（大动作 → 精细动作 → 口腔动作 → 动作序列 → 同伴模仿）
  ↓
配对（物-物 → 图-图 → 图-物 → 颜色 → 形状 → 字母/数字 → 分类）
  ↓
指令跟随（单步 → 含动词+名词 → 2步 → 3步 → 含方位词 → 条件指令）
```

### ✋ 精细动作领域
```
抓握（整手抓 → 掌心抓 → 三指捏）
  ↓
捏取（大物品 → 小珠子 → 夹子/镊子操作）
  ↓
穿珠（大珠大绳 → 小珠细绳 → 按颜色/形状序列穿）
  ↓
控笔（点 → 横线短 → 横线长 → 竖线 → 圆 → 斜线 → 综合图形）
  ↓
剪纸（剪开口 → 剪直线 → 剪曲线 → 剪圆 → 剪复杂图形）
  ↓
书写（描红 → 仿写 → 独立书写 → 书写姓名 → 抄写句子）
```

### 🤝 社交领域
```
眼神接触（应答性注视 → 自发注视 → 维持注视3秒）
  ↓
共同关注-应答（跟随指向看 → 跟随眼神看 → 对他人展示有反应）
  ↓
共同关注-发起（指向分享 → 展示物品给他人 → 用语言引起注意）
  ↓
平行游戏（同一区域 → 同一材料 → 观察同伴 → 模仿同伴）
  ↓
轮流（物品轮流 → 游戏轮流 → 对话轮流）
  ↓
合作游戏（共同目标 → 分工合作 → 协商规则）
  ↓
假想游戏（单一动作 → 2步序列 → 角色扮演 → 共建场景 → 社会戏剧游戏）
  ↓
会话技能（回应问题 → 发起话题 → 维持话题4轮+ → 话题切换 → 修复沟通断裂）
```

### 🔢 认知/学业预备领域
```
配对（实物 → 图片 → 颜色 → 形状 → 大小 → 多维度）
  ↓
分类（按类别 → 按功能 → 按特征 → 多维度分类）
  ↓
数概念（唱数 → 点数1-5 → 点数1-10 → 点数1-20 → 按数取物 → 数字认读 → 数字书写）
  ↓
因果关系（简单因果配对 → 看图说因果 → 预测结果 → 解释原因）
  ↓
序列（2步序列 → 3步序列 → 故事排序 → 时间顺序）
```

## 四、年龄发育里程碑速查表

> 以典型发展儿童为参照，帮助判断个案当前能力所处的发展年龄段。
> ⚠️ 仅作参考，不作为教学目标直接依据——需结合个案实际能力和 IEP 目标。

### 2 岁
| 领域 | 典型能力 |
|:---|:---|
| 语言理解 | 理解50+词，执行简单指令 |
| 语言表达 | 使用50+词，开始2词组合 |
| 精细动作 | 叠2-3块积木，翻书页 |
| 社交 | 平行游戏，开始表现所有权 |
| 认知 | 形状嵌入板，简单配对 |

### 3 岁
| 领域 | 典型能力 |
|:---|:---|
| 语言理解 | 理解2步指令，辨别大/小 |
| 语言表达 | 3-4词句子，回答简单WH问题 |
| 精细动作 | 穿大珠，剪开口，画圆 |
| 社交 | 开始假想游戏，简单轮流 |
| 认知 | 配对颜色/形状，点数1-3 |

### 4 岁
| 领域 | 典型能力 |
|:---|:---|
| 语言理解 | 理解3步指令，辨别颜色/形状名称 |
| 语言表达 | 5-6词句子，讲述简单经历，使用因为/所以 |
| 精细动作 | 剪直线，画十字/方形，扣纽扣 |
| 社交 | 合作游戏，理解轮流规则，假想游戏2-3步 |
| 认知 | 点数1-10，按数取物，简单分类 |

### 5 岁
| 领域 | 典型能力 |
|:---|:---|
| 语言理解 | 理解时间概念（先/后），条件指令 |
| 语言表达 | 复杂句子，讲述完整故事（有开头中间结尾） |
| 精细动作 | 剪曲线，描写笔画，写自己名字 |
| 社交 | 理解简单社交规则，协商，较复杂角色扮演 |
| 认知 | 认读数字1-20，简单加减概念，序列排列 |

### 6 岁
| 领域 | 典型能力 |
|:---|:---|
| 语言理解 | 理解被动句、比喻，听故事回答推理问题 |
| 语言表达 | 使用复合句，描述问题解决过程 |
| 精细动作 | 独立书写常用字，剪复杂图形，系鞋带 |
| 社交 | 有稳定玩伴，理解他人感受（初步心智理论） |
| 认知 | 20以内加减，分类计数，简单推理 |

### 7 岁
| 领域 | 典型能力 |
|:---|:---|
| 语言理解 | 理解讽刺/幽默，多步复杂指令 |
| 语言表达 | 有组织地叙述事件，使用连接词 |
| 精细动作 | 流畅书写，手工制作，精确控笔 |
| 社交 | 理解社交暗示，团队合作，冲突解决 |
| 认知 | 100以内数概念，简单应用题，时间/货币概念 |
