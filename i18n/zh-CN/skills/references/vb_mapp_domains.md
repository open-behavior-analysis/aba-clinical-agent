# VB-MAPP 评估领域标准参考字典 (Reference only)
本文件包含 VB-MAPP 评估系统中的标准技能领域（Domains）。
当执行 `assessment-logger` 生成评估报告表格时，请阅读本字典作为填充“领域”字段的标准词汇库。

## 一、 里程碑评估 (Milestones Assessment) 领域

| 领域 (Domain) | 核心定义简述 |
| :--- | :--- |
| **Mand (要求)** | 出于动机（MO）发起的主动索要行为。 |
| **Tact (命名)** | 看到/听到事物后引发的客观命名行为。 |
| **Listener Responding (听者反应)** | 听懂指令并作出相应的动作反应。 |
| **VP-MTS (视觉感知与样本配对)** | 从简单的视觉追踪到复杂的分类和拼图配对。 |
| **Independent Play (独立游戏)** | 在无成人干预下独自接触玩具的时间和方式。 |
| **Social Behavior (社交行为)** | 与成人或同伴发起的互动、平行游戏或合作游戏。 |
| **Motor Imitation (动作模仿)** | 模仿他人大类或精细运动的能力。 |
| **Echoic (仿说/复诵)** | 听到声音后进行精确的语音复述。 |
| **Vocal Behavior (发音行为)** | 自发发出的语音、词汇数量。 |
| **LRFFC (依功能特征分类的听者反应)** | 听到物体的功能/特征描述后指出该物体。 |
| **Intraverbal (交互式语言)** | 回答问题或接话茬（如：天黑了要...睡觉）。 |
| **Classroom Routines (集体常规)** | 在小组或教室环境中跟随集体指令的能力。 |
| **Grammar (语法)** | （阶段3）使用复数、时态等复杂语法。 |
| **Reading (阅读)** | （阶段3）字母识别与早期认字。 |
| **Writing (书写)** | （阶段3）控笔与早期书写。 |
| **Math (数学)** | （阶段3）早期的数感、数数和对应。 |

## 二、 障碍评估 (Barriers Assessment) 常见核心障碍项

当生成报告的“障碍评估”部分时，可参考以下高频维度词汇：
- **负向行为 (Negative Behaviors)**
- **指令控制/逃避 (Instructional Control / Escape)**
- **强化物受限 (Absent, Weak, or Impaired MO)**
- **发音不良 (Defective Articulation)**
- **自我刺激行为 (Self-Stimulation)**
- **过度依赖辅助 (Prompt Dependency)**
- **猜测/冲动 (Guessing Responses)**
- **泛化失败 (Failure to Generalize)**
